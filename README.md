## 내장함수

- isupper(): 대문자인지 확인
- islower(): 소문자인지 확인
- ord(): 아스키넘버로 변환
- sorted(): 정렬된 리스트 반환 / x=sorted(x,reverse=True) reverse=True이면 내림차순, reverse=False면 오름차순
- sort(): 리스트 자체를 정렬(메소드지만 비교) / x.sort(reverse=True) reverse=True이면 내림차순, reverse=False면 오름차순
